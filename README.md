# .github

Public repo for our Organisation

`/profile/README.md` will be shown as our Organisation's profile.

Issue templates *must* be located in the `.github/ISSUE_TEMPLATE` folder but all others can be in

1. Project root
2. `.github` folder
3. `docs` folder
